# 🎮 Advanced Tic Tac Toe (Tkinter GUI)

A modern, animated Tic Tac Toe game built with Python using Tkinter.

## ✨ Features

* 🎨 Premium dark UI
* 🤖 AI with 3 difficulty levels:

  * Easy (Random)
  * Medium (Smart + Random)
  * Hard (Minimax AI)
* 👥 2 Player Mode
* 📊 Scoreboard tracking
* ⚡ Smooth animations

## 🧠 AI Logic

Uses the **Minimax Algorithm** to make optimal moves in Hard mode.

## 🚀 How to Run

```bash
git clone https://github.com/your-username/tic-tac-toe.git
cd tic-tac-toe
python main.py
```
